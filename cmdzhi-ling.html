CMD指令

要登入ec2要有key(private key, 副檔名為.pem)

  • ssh -i <private key> ec2-user@<ip>

本地端若要傳檔至ec2

  • scp -r -i <private key> <local file path> ec2-user@<ip>:/home/ec2-user/

    • (/home/ec2-user為存放位置)
